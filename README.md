# Sales-Profit-Analysis-Report-Using-Power-BI
Built an interactive Power BI dashboard to analyze sales and profit performance across categories, markets, and customer segments. The dashboard features KPI tracking, sales trend analysis, customer insights, interactive filtering, and drill-through functionality for detailed data exploration and data-driven decision-making.

Key Features:
KPI Cards for Sales and Profit Tracking
Year-wise Sales Trend Analysis
Category-wise Sales Distribution
Market and Region Performance Analysis
Top 5 Customers by Sales Revenue
Interactive Filters and Drill-Through Functionality for Detailed Insights
